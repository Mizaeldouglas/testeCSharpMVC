## Stack utilizada

**Back-end e Front-end:** C#, Dotnet 7


## Instalação

Clonado projeto:

```bash
  git clone https://github.com/Mizaeldouglas/testeCSharpMVC.git
  cd testeCSharpMVC
```

Abre o projeto numa IDE ou no terminal Mesmo:

```bash
  dotnet restore
  dotnet build
  dotnet run
```
